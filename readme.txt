About:
Simulation game for MR estimates and virtual intervention propagation simulation
Game GUI currently supports up to 10 policies and 10 goals



#### .bat not included in current release, but instructions kept for reference in how one might modify gamedata ####
Instructions:
To run the game:
    1) Simply run run.bat
To modify gamedata:
    1) Read read-me's in recompilers folder
    2) Read read-me's in re/compiler sub-folders (i.e., MR, MRNV & game recompilers)
    3) Run new MR analyses using MR compiler as appropriate
    4) Change options as appropriate in recompiler sub-folders
    5) Return to the main directory and run recompile.bat to run all of the recompilers
    6) Run the game as normal and recompiled data will be loaded
